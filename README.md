# MahirTrade Landing Page

Website landing page untuk komunitas MahirTrade (Retail & Funded Forex Community).

## Struktur Project

Project ini menggunakan struktur static website standar untuk deployment di shared hosting (cPanel/Niagahoster).

```
public_html/
├── index.html          # Halaman utama
├── assets/
│   ├── css/
│   │   └── style.css   # Styling utama
│   ├── js/
│   │   └── main.js     # Script interaktif
│   ├── img/            # Gambar (Hero, dll)
│   └── icons/          # Icon SVG
└── README.md
```

## Cara Deployment

1.  Login ke cPanel hosting Anda.
2.  Buka **File Manager**.
3.  Masuk ke folder `public_html`.
4.  Upload semua file dan folder yang ada di dalam direktori `public_html` repository ini.
5.  Website siap diakses.

## Teknologi

*   HTML5
*   CSS3 (Vanilla)
*   JavaScript (Vanilla)
*   Google Fonts (Inter)

## Catatan

*   Tidak memerlukan `npm install` atau build process.
*   Edit `index.html` untuk mengubah konten.
*   Edit `assets/css/style.css` untuk mengubah tampilan.
